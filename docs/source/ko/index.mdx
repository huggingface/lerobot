# LeRobot

**실세계 로보틱스를 위한 최첨단 머신러닝**

🤗 LeRobot은 PyTorch 기반의 실세계 로보틱스를 위한 모델, 데이터셋, 도구를 제공하는 것을 목표로 합니다. 로보틱스 진입 장벽을 낮추어 누구나 데이터셋과 사전학습된 모델을 공유하고 활용할 수 있도록 하는 것이 목표입니다.

🤗 LeRobot은 모방 학습과 강화 학습에 초점을 맞춘, 실세계 환경으로의 전이가 검증된 최첨단 접근법들을 포함하고 있습니다.

🤗 LeRobot은 사전학습된 모델, 사람이 수집한 시연 데이터셋, 시뮬레이션 환경을 이미 제공하고 있어 누구나 바로 시작할 수 있습니다.

🤗 LeRobot은 사전학습된 모델과 데이터셋을 [LeRobot HuggingFace 페이지](https://huggingface.co/lerobot)에서 호스팅하고 있습니다.
