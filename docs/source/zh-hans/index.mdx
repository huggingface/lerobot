# LeRobot

**面向现实世界机器人的最先进的机器学习技术**

🤗 LeRobot的目标是为现实世界机器人提供基于PyTorch的模型、数据集、以及操作工具。我们的目标是降低学习机器人的准入门槛，让每一个人都可以参与到机器人数据集和预训练模型的开源运动中，并且从中受益。

🤗 LeRobot实现了许多可用于现实世界的最先进的模仿学习，强化学习算法。

🤗 LeRobot已经提供了一套让每个人都可以轻松开始的机器人预训练模型、基于人类演示的机器人数据集、以及仿真环境。

🤗 LeRobot将这套预训练模型和数据集存储在[LeRobot HuggingFace](https://huggingface.co/lerobot)页面上。
