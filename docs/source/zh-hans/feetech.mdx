# 升级飞特舵机固件

这篇教程将会指导你使用飞特官方调试软件升级飞特舵机的固件。

# 前置要求

- Windows操作系统电脑 (飞特调试软件只可以在Windows平台运行)
- 一个飞特舵机控制板
- 一条USB线材用于电脑与舵机控制板的连接
- 将飞特舵机连接到控制板上

## 第一步： 下载飞特调试软件

1. 进入飞特官方软件下载页面: [https://www.feetechrc.com/software.html](https://www.feetechrc.com/software.html)
2. 下载最新版本的飞特调试软件 (FD)
3. 将调试软件安装到Windows电脑上

## 第二步： 配置硬件

1. 将飞特舵机连接到舵机控制板上
2. 通过USB线材连接电脑与舵机控制板
3. 确保电源适配器对舵机的供电(舵机背部红灯亮起)

## 第三步： 配置电脑与舵机的通信

1. 打开飞特调试软件
2. 在 通信设置-端口号 的下拉菜单中选择正确的COM端口。
   - 如果不确定使用哪一个端口号，检查Windows设备管理器中的 “端口 (COM & LPT)”
3. 设置适当的波特率 (对于大多数飞特舵机来说1000000波特率是典型值)
4. 点击 “打开” 来建立调试软件与控制板的通信

## 第四步： 舵机搜索

1. 一旦连接建立，点击 “搜索” 按钮去探测所有已连接在控制板上的舵机
2. 调试软件会自动发现并且列出所有在串行总线上的舵机
3. 每个舵机通过 ID号与型号 的方式列出

## 第五步： 升级固件

对于要更新的每个舵机：

1. **选择舵机** 点击在列表中的目标舵机
2. **点击“升级“选项卡**:
3. **点击“在线检测”按钮**:
   - 如果发现可能的固件更新，将显示在按钮上侧的框中
4. **点击升级按钮**:
   - 将显示升级进度

## 第六步： 确认固件更新

1. 更新完成后，软件应自动刷新舵机的信息
2. 确认固件版本已更新到预期版本

## 注意事项

⚠️ **警告**： 不要在固件更新期间断开电源或USB，它可能会使舵机损坏

## 奖励：在Linux/macOS调试舵机

如果只是想调试舵机，你可以使用开源的Feetech调试工具：

- **存储仓库**: [FT_SCServo_Debug_Qt](https://github.com/CarolinePascal/FT_SCServo_Debug_Qt/tree/fix/port-search-timer)

### 安装说明

请按照存储仓库中的说明进行工具的安装，Ubuntu可以直接安装，MacOS需要从源代码构建。

**限制：**

- 此工具仅用于调试和参数调整
- 固件更新仍然须在Windows上使用官方Feetech调试软件进行
