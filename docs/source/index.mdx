# LeRobot

**실제 로봇을 위한 최첨단 머신러닝**

🤗 LeRobot은 PyTorch에서 실제 로봇을 위한 모델, 데이터셋 및 도구를 제공하는 것을 목표로 합니다. 목표는 로봇 공학에 대한 진입 장벽을 낮춰 모든 사람이 데이터셋과 사전 학습된 모델을 공유하고 기여하며 혜택을 받을 수 있도록 하는 것입니다.

🤗 LeRobot은 모방 학습과 강화 학습에 중점을 두고 실제 환경으로 전환된 것으로 입증된 최첨단 접근 방식을 포함하고 있습니다.

🤗 LeRobot은 이미 모든 사람이 시작할 수 있도록 사전 학습된 모델, 사람이 수집한 시연이 포함된 데이터셋, 시뮬레이션 환경을 제공합니다.

🤗 LeRobot은 LeRobot HuggingFace 페이지에서 사전 학습된 모델과 데이터셋을 호스팅합니다.
