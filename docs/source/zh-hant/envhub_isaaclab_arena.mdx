# NVIDIA IsaacLab Arena & LeRobot

LeRobot EnvHub 現已支援透過 IsaacLab Arena 進行 **GPU 加速模擬（GPU-accelerated simulation）**，可大規模進行策略評估（policy evaluation at scale）。
使用高擬真模擬訓練（high-fidelity simulation training）與評估模仿學習策略（evaluate imitation learning policy）——全部整合至 LeRobot 生態系統（ecosystem）中。

[IsaacLab Arena](https://github.com/isaac-sim/IsaacLab-Arena) 與 NVIDIA IsaacLab 整合，提供：

- 🤖 **人形機器人實體（humanoid embodiments）**：GR1、G1、Galileo 及各種配置（configurations）
- 🎯 **操作與移動操作任務（manipulation & loco-manipulation tasks）**：開門、取放物品、按按鈕等
- ⚡ **GPU 加速 rollouts（GPU-accelerated rollouts）**：在 NVIDIA GPU 上平行執行環境（parallel environment execution）
- 🖼️ **RTX 渲染（RTX rendering）**：以逼真的渲染（realistic rendering）、反射（reflections）與折射（refractions）效果評估基於視覺的策略（evaluate vision-based policies）
- 📦 **LeRobot 相容資料集（LeRobot-compatible datasets）**：可直接用於 GR00T N1x、PI0、SmolVLA、ACT 和 Diffusion 策略（policies）的訓練（training）
- 🔄 **EnvHub 整合（EnvHub integration）**：一行程式碼即可從 HuggingFace EnvHub 載入環境（environments）

## 安裝（installation）

### 先決條件（prerequisites）

硬體需求與 Isaac Sim 相同，詳見 [Isaac Sim 系統需求（Isaac Sim Requirements）](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/installation/requirements.html)。

- 支援 CUDA 的 NVIDIA GPU
- 與 IsaacSim 5.1.0 相容的 NVIDIA 驅動程式（driver）
- Linux（Ubuntu 22.04 / 24.04）

### 設定（setup）

```bash
# 1. 建立 conda 環境
conda create -y -n lerobot-arena python=3.11
conda activate lerobot-arena
conda install -y -c conda-forge ffmpeg=7.1.1

# 2. 安裝 Isaac Sim 5.1.0
pip install "isaacsim[all,extscache]==5.1.0" --extra-index-url https://pypi.nvidia.com

# 接受 NVIDIA EULA（必要）
export ACCEPT_EULA=Y
export PRIVACY_CONSENT=Y

# 3. 安裝 IsaacLab 2.3.0
git clone https://github.com/isaac-sim/IsaacLab.git
cd IsaacLab
git checkout v2.3.0
./isaaclab.sh -i
cd ..

# 4. 安裝 IsaacLab Arena
git clone https://github.com/isaac-sim/IsaacLab-Arena.git
cd IsaacLab-Arena
git checkout release/0.1.1
pip install -e .
cd ..


# 5. 安裝 LeRobot（evaluation extra 用於環境/策略評估）
git clone https://github.com/huggingface/lerobot.git
cd lerobot
pip install -e ".[evaluation]"
cd ..


# 6. 安裝額外依賴項
pip install onnxruntime==1.23.2 lightwheel-sdk==1.0.1 vuer[all]==0.0.70 qpsolvers==4.8.1
pip install numpy==1.26.0 # Isaac Sim 5.1 依賴 numpy==1.26.0，這將在下一版修正
```

## 環境配置（environment configuration）

### 完整配置選項（full configuration options）

```python
from lerobot.envs.configs import IsaaclabArenaEnv

config = IsaaclabArenaEnv(
    # 環境選擇
    environment="gr1_microwave", # 任務環境
    embodiment="gr1_pink", # 機器人實體
    object="power_drill", # 要操作的物件

    # 模擬設定
    episode_length=300,               # 每個回合的最大步數
    headless=True,                    # 無 GUI 模式執行
    device="cuda:0",                  # GPU 裝置
    seed=42,                          # 隨機種子

    # 觀測設定
    state_keys="robot_joint_pos",     # 狀態觀測鍵（以逗號分隔）
    camera_keys="robot_pov_cam_rgb",  # 相機觀測鍵（以逗號分隔）
    state_dim=54,                     # 預期狀態維度
    action_dim=36,                    # 預期動作維度
    camera_height=512,                # 相機影像高度
    camera_width=512,                 # 相機影像寬度
    enable_cameras=True,              # 啟用相機觀測

    # 影片錄製
    video=False,                      # 啟用影片錄製
    video_length=100,                 # 每段影片的幀數
    video_interval=200,               # 錄製間隔步數

    # 進階
    mimic=False,                      # 啟用模仿模式
    teleop_device=None,               # 遙操作裝置
    disable_fabric=False,             # 停用 fabric 最佳化
    enable_pinocchio=True,            # 啟用 Pinocchio 進行逆運動學
)
```

### 直接使用 Environment Hub 進行進階用途（using Environment Hub directly for advanced usage）

建立一個名為 `test_env_load_arena.py` 的檔案，或[從 EnvHub 下載](https://huggingface.co/nvidia/isaaclab-arena-envs/blob/main/tests/test_env_load_arena.py)：

```python
import logging
from dataclasses import asdict
from pprint import pformat
import torch
import tqdm
from lerobot.configs import parser
from lerobot.configs.eval import EvalPipelineConfig


@parser.wrap()
def main(cfg: EvalPipelineConfig):
    """對 IsaacLab Arena 環境執行隨機動作推演。"""
    logging.info(pformat(asdict(cfg)))

    from lerobot.envs import make_env

    env_dict = make_env(
        cfg.env,
        n_envs=cfg.env.num_envs,
        trust_remote_code=True,
    )
    env = next(iter(env_dict.values()))[0]
    env.reset()
    for _ in tqdm.tqdm(range(cfg.env.episode_length)):
        with torch.inference_mode():
            actions = env.action_space.sample()
            obs, rewards, terminated, truncated, info = env.step(actions)
            if terminated.any() or truncated.any():
                obs, info = env.reset()
    env.close()


if __name__ == "__main__":
    main()
```

執行方式（run with）：

```bash
python test_env_load_arena.py \
    --env.environment=g1_locomanip_pnp \
    --env.embodiment=gr1_pink \
    --env.object=cracker_box \
    --env.num_envs=4 \
    --env.enable_cameras=true \
    --env.seed=1000 \
    --env.video=true \
    --env.video_length=10 \
    --env.video_interval=15 \
    --env.headless=false \
    --env.hub_path=nvidia/isaaclab-arena-envs \
    --env.type=isaaclab_arena
```

## 建立新環境（creating new environments）

首先按照 [IsaacLab Arena 文件](https://isaac-sim.github.io/IsaacLab-Arena/release/0.1.1/index.html) 建立新的 IsaacLab Arena 環境（environment）。

複製我們的 EnvHub 儲存庫（repository）：

```bash
git clone https://huggingface.co/nvidia/isaaclab-arena-envs
```

根據您的新環境修（new environment）改 `example_envs.yaml` 檔案。
將修改後的儲存庫（repository）[上傳](./envhub#%E6%AD%A5%E9%A9%9F-3%E4%B8%8A%E5%82%B3%E5%88%B0-hubupload-to-the-hub)至 HuggingFace EnvHub。

  repository） 或者您也可以將環境程式碼和資產（assets）直接打包在 EnvHub
  儲存庫（repo）中。

然後在評估（evaluating）時使用您的新環境（new environment）：

```bash
lerobot-eval \
    --env.hub_path=<your-env-hub-path>/isaaclab-arena-envs \

```

## 疑難排解

### CUDA 記憶體不足（CUDA out of memory）

減少 `batch_size` 或使用擁有更多 VRAM 的 GPU：

```bash
--eval.batch_size=1
```

### EULA 未接受（EULA not accepted）

執行前設定環境變數（environment variables）：

```bash
export ACCEPT_EULA=Y
export PRIVACY_CONSENT=Y
```

### 影片錄製無法運作（video recording not working）

以無頭模式（headless）執行時啟用攝影機（enable cameras）：

```bash
--env.video=true --env.enable_cameras=true --env.headless=true
```

### 策略輸出維度不匹配（policy output dimension mismatch）

確保 `action_dim` 與您的策略（policy）匹配：

```bash
--env.action_dim=36
```

### Isaac Sim 初始化期間出現 libGLU.so.1 錯誤

確保已安裝以下依賴（dependencies），這在無頭機器（headless machines）上較容易發生。

```bash
sudo apt update && sudo apt install -y libglu1-mesa libxt6
```
