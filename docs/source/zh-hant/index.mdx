# LeRobot

**用於真實世界機器人的最先進機器學習技術**

🤗 LeRobot 旨在以 PyTorch 打造適用於真實世界機器人技術的模型（model）、資料集（dataset）與工具。目標是降低機器人技術的入門障礙，讓所有人都能透過共享資料集與預訓練模型（pretrained model）來貢獻心力並從中受益。

🤗 LeRobot 包含了已證實可應用於真實世界的最先進方法，並聚焦於模仿學習（imitation learning）與強化學習（reinforcement learning, RL）。

🤗 LeRobot 提供一系列預訓練模型（pretrained model）、透過人工示範蒐集的資料集（dataset）以及模擬環境（simulation environments），讓所有人都能開始入門。

🤗 LeRobot 將預訓練模型（pretrained model）與資料集（dataset）託管於 LeRobot HuggingFace 頁面。
